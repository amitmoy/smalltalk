8 queens problem graphical solution, by: Amit Moyal for course: Object Oriented Programming 
in Open University of Israel.

To run the graphical 8 queens problem solutions do the next steps:
1.copy queen.png to the resources folder
2.run squeek
3.open workspace -> board:=Board new. board openInWorld.
4.put the mouse pointer above the chess board, click the right arrow key to get the next solution.
